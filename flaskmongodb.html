Flask ile MongoDB Örneği

Posted on Pzt 12 Ağustos 2019 in Flask

Merhaba arkadaşlar, bu yazımda Flask ve MongoDB kullanarak veritabanına bilgi ekleyeceğiz.

Anlatımım Ubuntu 19.04 üzerind gerçekleşecektir

İlk olarak neyin ne olduğundan bahsedelim. Sonrasında gerekli kurulumları yapalım ve ardından hızlıca kodumuza geçelim.

MongoDB Nedir?

MongoDB, NoSQL verilerini belge biçiminde saklayıp, analiz edebileceğimiz ölçeklenebilir bir veritabanı teknolojisidir. MongoDB özgür yazılım lisansı ile yayınlanmış açık kaynaklı bir teknolojidir.

  • Ölçeklenebilirdir (Scalable). Veri boyutu arttığı durumlarda veya performans sıkıntısı yaşadığımız durumlarda makine ekleyebiliriz
  • Veriler belge biçiminde saklanır. Burada JSON verilerini kullanabiliriz
  • Veriler JSON şeklinde saklandığı için gelen veri yapısı değişse bile kaydetme işleminde sıkıntı yaşanmaz.
  • Verilerin birden fazla kopyası saklanabilir ve veri kaybı yaşanmaz.

Flask Nedir?

Herkese merhaba. Flask bildiğiniz gibi python ile yazılmış bir microframework (mikro çatı). Oluşturacağım küçük web projelerini flask ile yapmayı tercih ediyorum. Ancak daha karmaşık projeleri küçük küçük kütüphaneler kurarak kolayca yapmanız mümkün. Flask ile daha detaylı araştırma yapabilirsiniz.

Gerekli Modüller

Aşağıdaki modülleri kurmamız yeterli olacaktır.

pip3 install Flask

pip3 install Flask-PyMongo

Gerekli Ayarlar

MongoDB Atlas üzerinden kendinize bir free cluster oluşturmalısınız. Yoksa kodumuzdaki [MONGO_URI] kısmıyla ilgili sorun yaşayabiliriniz. Şimdi gelelim kodumuza.

app.py

from flask import Flask, render_template
from flask_pymongo import PyMongo

app = Flask(__name__)
app.config['SECRET_KEY'] = 'linuxdegilgnulinux'
app.config[
    'MONGO_URI'] = "mongodb://<username>:<password>@myflask-shard-00-00-raeh0.mongodb.net:27017,myflask-shard-00-01-raeh0.mongodb.net:27017,myflask-shard-00-02-raeh0.mongodb.net:27017/test?ssl=true&replicaSet=myFlask-shard-0&authSource=admin&retryWrites=true&w=majority"
mongo = PyMongo(app)

@app.route('/')
def home():
    return render_template("index.html")


@app.route('/create', methods=['POST'])
def create():
    username = request.form.get('username')
    mongo.db.user.insert_one({'username': username})

    return 'Done!'

if __name__ == '__main__':
    app.run(debug=True)
